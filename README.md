
# 🎬 Netflix Data Analysis (EDA)

This project performs **Exploratory Data Analysis (EDA)** on a Netflix dataset to understand content trends, distribution, and key insights.

---

## 📊 Objectives

* Analyze Netflix content (Movies & TV Shows)
* Explore trends by year, genre, and country
* Identify most common ratings and categories
* Understand content growth over time

---

## 🛠 Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 📁 Dataset

* Netflix dataset (titles, genres, ratings, release year, etc.)
* Includes Movies and TV Shows

---

## 🔍 Key Analysis

* Content distribution (Movies vs TV Shows)
* Top genres on Netflix
* Year-wise content growth
* Most frequent ratings
* Country-wise content production

---

## 📈 Visualizations

* Bar charts
* Count plots
* Distribution plots
* Trend analysis graphs

---
